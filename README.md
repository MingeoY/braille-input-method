# 13键盲文输入法

## 项目简介

13键盲文输入法是一款基于Web的多语言盲文输入解决方案，通过创新的13键键盘布局实现高效的盲文输入，支持多种语言的实时转换显示。

## 6点盲文与13键映射

### 盲文结构
6点盲文由2列3行盲文点位构成，包含空方(⠀)共64个盲符，按照规律分为5行：
- 第一行的3、6点位均为空，
- 第二行在第一行的基础上增加3点位，
- 第三行在第一行的基础上增加3、6点位，
- 第四行在第五行的基础上增加6点位，
- 第五行将第一行由1、2、4、5点位整体下移至2、3、5、6点位（空方下移不变）。

## 多语言输入功能

### 核心特性
- **无需切换输入法**：通过盲文中介，同时显示多种语言字符
- **实时转换**：输入盲文即时转换为各语言文本
- **多语言支持**：世界语、汉语注音符号、英语、法语、俄语等
- **灵活配置**：最多支持10种语言同时显示

### 界面布局

| 键盘输入               | 文本输出                       |
|-----------------------|--------------------------------|
| 3×5键盘               | 盲文显示框(复制、删除、清空)      |
| 输入方式介绍和操作演示  | 语种1显示框(语音提示、复制、移除) |
| 盲文键盘示意图         | 语种2显示框(复制、移除)          |
|                       | 添加更多语种(最多10个)          |

各语种显示框由标题、文本框、复制键组成，盲文显示框还包括删除键和清空键。

具体到各语种显示，如果某个盲符暂无对应的字符，就在语种显示框中用红色盲符显示。

默认状态不出现语种显示框，用户可选择增加语种、移除语种框或调换语种顺序，添加的语种可重复，但最多同时显示10个。

键盘显示键：按下盲符时，除了显示当前输入的盲符外，还应显示该盲符在语种1中代表的字符。键盘13键在中央和四周显示点击和滑动输入的盲符在语种1中的代表字符，按下时对应方向的字符放大。

### 汉语注音符号
支持完整的注音符号系统，包括：
- **声调**：阴平、阳平、上声、去声（分别标ˉˊˇˋ：标准的注音符号阴平不标调，这里为简化实现所以阴平标ˉ）
- **声母变音规则**：g/k/h在i/ü前变为j/q/x
- **韵母变音规则**：o在b/p/m/f后变为e

## 语音提示功能

### 支持语言
- ✅ 汉语
- ✅ 英语  
- ✅ 俄语
- ✅ 日语
- ❌ 世界语（不支持）
- ❌ 法语（不支持）

### 语音提示内容
- **中文**：按注音符号对应的汉字读音
- **英语**：按字母名称读音
- **俄语**：按字母名称读音
- **日语**：按假名读音
- **标点符号**：各语言对应的标点名称

## 界面语言支持

支持6种网页界面语言：
- 中文 (zh-CN)
- English (en)
- русский (ru)
- français (fr)
- español (es)
- العربية (ar)

## 技术特性

### 输入方式
- **点击输入**：中央字符
- **滑动输入**：四个方向对应不同字符
- **触摸优化**：支持移动设备触摸操作
- **光标控制**：通过控制键滑动移动文本光标

### 交互功能
- **拖拽排序**：语言显示框可拖拽重新排序
- **实时预览**：显示键实时显示当前输入字符
- **一键操作**：复制、删除、清空功能
- **宽度调整**：输入区域宽度可拖动调整

## 使用说明

1. **选择语言**：点击"添加语种"选择需要的语言
2. **开始输入**：通过点击或滑动键盘输入盲文
3. **查看结果**：各语言显示框实时显示转换结果
4. **管理内容**：使用复制、删除、清空功能管理文本
5. **调整界面**：拖拽调整语言顺序或输入区域宽度

## 部署要求

- 现代浏览器支持（Chrome、Firefox、Safari、Edge）
- 支持Web Speech API（语音功能）
- 支持Clipboard API（复制功能）
- 响应式设计，支持桌面和移动设备
